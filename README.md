# recruitment_email_organization
利用coze整理邮箱中的招聘邮件,在飞书多维表格和日历中显示
这个项目我的第一个coze工作流项目，他的完成度可能没有那么高，但是可以为大家的找工作进行一个还算可以的记录

##该项目主要有三个大部分组成
1.coze工作流
2.飞书多维表格与飞书日历
3.飞书工作流

##下面简述三个部分的主要功能
#coze工作流
完成qq邮箱中招聘相关邮件的读取、整理，最后将结论输出到飞书的多维表格与日历，完成招聘流程的记录并且在日历中提醒完成招聘流程
#飞书多维表格与飞书日历
飞书多维表格用来记录整个面试流程的变更，飞书日历中设置提醒，提醒完成相关流程
#飞书工作流
飞书工作流作为一个定时触发器，触发coze工作流在每天的固定时间完成一次更迭
